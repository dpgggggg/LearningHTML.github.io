Steps in bulidng an Image
  • Pick and Image (careful of the file extension in the URL)
  • Record the URL, cite it, comment for the date accessed, and record the original dimensions
  • Note: opening in a new tab will eventually provide a http or https URL with a file extension
  • Use blockquote and comment as appropriate
  • Use img-tag with attribute src, alt, title, and style with width-height
